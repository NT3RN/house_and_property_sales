# Property Sales Data Science Project

## Description
This repository contains a full exploratory data analysis (EDA) on a Property Sales dataset using Python. The project focuses on understanding sales trends, price patterns, bedroom distribution, property types, and year-wise comparisons through structured data processing and visualization.

## Features
This project performs:

- Importing required Python libraries  
- Reading and loading the Property Sales dataset  
- Calculating:  
  - Total number of houses and properties sold  
  - Average house price  
  - Highest sales value  
- Grouping the dataset by:  
  - Year  
  - Number of bedrooms  
  - Property type  
- Extracting targeted insights:  
  - Cheapest house sale in **2010**  
  - Most expensive house sale in **2017**  
  - Most expensive house with **5 bedrooms**  
  - Cheapest unit overall  
  - Cheapest unit in **2008**  
  - Average house price in **2015**  
  - Total unit sales in **2016**  
  - Cheapest house sale at address/code **2617**  
- Visualizations:  
  - Pie chart of sales distribution by number of bedrooms  
  - Histogram showing full sales distribution  

## Technologies Used
- Python 3  
- pandas  
- matplotlib
